This workshop introduces you the VSCode debugger and the common workflow to debug a NodeJS application.

## What You Will Learn

 - Starting a Debug Session: via Auto configuration or debug terminal

 - Breakpoints: Pause execution as you like

 - Logpoints: Log messages without breaking

 - Debug Actions: Navigate through code execution

 - Debug Console REPL: interact with your code in real-time

 - Launch Scripts: Define & save how you want to launch your code

## Preparation

### Cloning the Repo

```sh
git clone git@github.com:Jkker/vscode-nodejs-debugging.git
cd vscode-nodejs-debugging
```

### Install Dependencies

```sh
npm install
```

### Install VSCode Plugins

 - [Rest Client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client)
 - [Turbo Console Log](https://marketplace.visualstudio.com/items?itemName=ChakrounAnas.turbo-console-log)
